# IP Network Mask Calculator

This project is a simple web application that demonstrates how IP Network Masks work. It allows users to input an IP address and a netmask, displays their binary representations, and calculates the network address using a logical AND operation.

## How to Run the Application

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Enter an IP address and a netmask in the provided input fields.
4. View the binary representations and the calculated network address displayed on the page.
